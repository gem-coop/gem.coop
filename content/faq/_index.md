## Is it planned to offer accounts similar to rubygems.org?

TBD, for now, we're focused on adding [namespaces](/updates/5/) which solve several problems when publishing gems.

## Is there a roadmap for gem.coop?

There's no roadmap yet. Recently, we introduced [gem cooldowns](/cooldowns/) which mitigate supply chain attacks by hides gems until they have been published for 48 hours.

## I have an idea/suggestion for gem.coop

If you have an idea or suggestion for improvements/new features then let us know using [GitHub Discussions](https://github.com/gem-coop/gem.coop/discussions).

## I found an issue

If there's no existing report on [GitHub](https://github.com/gem-coop/gem.coop/issues), then please file a [new one](https://github.com/gem-coop/gem.coop/issues/new).
